API Endpoints for Custom Bots

download POST /audio/download
Download multiple Roblox audio files at once. Supports up to 10 assets per request with real-time progress tracking.
Authentication: Bearer Token or API Key
Rate Limit: 100 requests/hour
info GET /audio/info
Get detailed information about a Roblox audio asset including name, creator, and description.
/info asset_id:123456789
layers /batch
Download consecutive audio files in batch. Perfect for downloading series of related audio assets.
/batch place_id:123456 start_id:1001 count:5
